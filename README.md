## Инструкция для пользователя
Сборка проекта производится следующим образом:

Необходимо создать директорию *build*
```console bash
mkdir -p build && cd build
```
Произвести сборку
```console bash
cmake .. && cmake --build .
```
Для запуска необходимо ввести:
```console bash
./<path>/recursive_encoder ${MODE} ${ENCODING_TARGET}
```
Описание передаваемых параметров:
* *MODE* - принимает значения
    1. encrypt - шифровать
    2. decrypt - дешифровать
* *ENCODING_TARGET* - путь до шифруемой/дешифруемой цели
